O que é Dropwizard?

O que é Dropwizard?

  • Coleção de bibliotecas
  • Glue code
  • Objetivo:
  • Criar aplicações(microservices) facilmente

Objetivos

  • Agilidade na criação de novos serviços
  • Simplicidade
  • Facilidade de deploy
  • Monitoramento

Optionated*

  • Jetty para HTTP
  • Jersey para REST (JAX-RS)
  • Jackson para JSON
  • Metrics para instrumentação
  • Logback para logging
  • Guava
  • Hibernate validator (JSR-303)
  • Apache HTTP Client

Modulos

  • dropwizard-core
  • dropwizard-auth
  • dropwizard-testing
  • dropwizard-hibernate
  • dropwizard-client
  • ...

Recursos extras

  • Bundles
  • Tasks
  • Commands
  • Testing
  • Admin server
  • Managed Objects

Aplicação minima

  • Maven
  • Application class
  • Configuration class
  • Representation class (Opcional)
  • Resource class (Opcional)
  • Healthcheck
  • Metricas customizadas
